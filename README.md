# GeekPulse Dashboard

Welcome to **GeekPulse Dashboard**—a gloriously geeky mashup of free PC games, the next MCU film countdown, and Hacker News stories, all in one chaotic, nerd-approved dashboard. Built with React.js, Tailwind CSS, and SWC-powered speed, this is my playground for trial, error, and celebrating the pulse of geekdom.

## Features

- **Free PC Games**: Browse free-to-play PC gems from the FreeToGame API.
- **MCU Countdown**: Count the days to the next Marvel Cinematic Universe blockbuster.
- **Hacker News Highlights**: Catch a slice of tech nerdery from Hacker News.

## Tech Stack

- **React.js**: Dynamic components and state wizardry.
- **JavaScript**: API fetching and nerdy logic.
- **SWC**: Blazing-fast compiles via Vite.
- **Tailwind CSS**: Instant, responsive styling with utility classes.
- **Vite**: The dev tool keeping it all snappy.

## APIs Used

| Feature       | API Endpoint                                           | Documentation                                                                           |
| ------------- | ------------------------------------------------------ | --------------------------------------------------------------------------------------- |
| Free PC Games | `https://www.freetogame.com/api/games?platform=pc`     | [FreeToGame](https://www.freetogame.com/api-doc)                                        |
| Next MCU Film | `https://www.whenisthenextmcufilm.com/api`             | [MCU Countdown API](https://github.com/DiljotSG/MCU-Countdown/blob/develop/docs/API.md) |
| Hacker News   | `https://hacker-news.firebaseio.com/v0/item/8863.json` | [Hacker News API](https://github.com/HackerNews/API)                                    |

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm (bundled with Node.js)

### Installation

1. Clone the repo:

```bash
git clone https://github.com/jacobsmxth/GeekPulseDashboard.git
cd geekpulse-dashboard
```

2. Install dependencies:

```bash
npm install
```

3. Fire up the dev server:

```bash
npm run dev
```

4. Hit http://localhost:5173 in your browser and soak in the geekiness!

## Notes

- This is a live experiment—bugs and brilliance guaranteed.
- APIs are public, no keys needed, but watch for rate limits.
